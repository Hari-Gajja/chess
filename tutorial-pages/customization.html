Board Customization

Color Settings

Customize your board colors:

  1. Use the color pickers below the board
  2. Select colors for light and dark squares
  3. Click "Apply Colors" to update the board
  4. Colors persist between games

Board Layout

Additional customization options:

  • Flip board orientation
  • Set custom piece positions
  • Adjust board size (responsive)
