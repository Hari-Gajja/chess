Game Controls

Control Buttons

  • Start Game: Begins a new chess match
  • Play Again: Resets the board for a new game
  • Set Position: Customize piece placement
  • Flip Board: Rotates the board 180 degrees

Timer Controls

The game features a 10-minute timer for each player:

  • Timer starts when the first move is made
  • Switches automatically between players
  • Game ends if time runs out
